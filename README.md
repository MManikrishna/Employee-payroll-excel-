Employee Payroll System (Excel)

Project Overview
This project is a simple Employee Payroll System built using Microsoft Excel for one month.

Purpose
To calculate employee salaries based on hours worked and hourly wages.

Features

Calculates total salary using formula

Organizes employee data in table format

Uses Excel formulas for automatic calculation

Demonstrates real-world payroll logic

Columns Used

Employee Name

Hours Worked

Hourly Wage

Total Pay

Net Pay
